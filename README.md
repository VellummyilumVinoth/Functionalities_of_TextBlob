# Functionalities_of_TextBlob
What is TextBlob
  * TextBlob is an open-source python library used to perform NLP activities like Lemmatization, Stemming, Tokeneization, Noun Phrase Extraction, POS, Tagging, N-Grams, Sentimental Analysis.
  * It is faster than NLTK, however it does not provide the functionalities like vectorization, dependency parsing.
  * Text Classification, Sentiment Analysis can be performed using TextBlob.
  * Installation: pip install textblob
